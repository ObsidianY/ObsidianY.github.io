各种错误

2020-12-30

对各种错误的总结。

本页面将持续更新

动态规划

  1. 单调队列 dp 弹出队尾 while(l <= r && q[l] < ...) ++l; 不是 while(l <= r && l < ...) ++l; !!!

数据结构

  1. $01trie$ 记得在算 $id$ 的时候加 bool

  2. 线段树 pushdown 是 Add 两个儿子（丢人

  3. 主席树要记得开大空间，维护多种值时别写混了。

图论

树

  1. 树上差分是 $l,r (+1);lca,fa[lca] (-1)$

网络流

  1. 记得注意初始化的范围，一定要跑满。
  2. 编号从 $2$ 开始！cnt 要设为 $1$ !!!

数学

数论

筛 $\mu$ 时把质数的 $\mu$ 设为了1。

多项式

zzm的两个FFT典型:

  1. w=(a,b)
  2. w((a,b))
