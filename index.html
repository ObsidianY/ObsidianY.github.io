2021-03-17
#考试 #dp #杂项 #分治 #二分

Review

这次考试大家都基本拿的是暴力分，除了几个神仙正解写挂了。

反思一下：

$\mathcal{T1}$ 想到了可删堆，还考虑了左偏树（傻了），但是这种奇妙的性质似乎难以挖掘，这种题只能看运气了。

$ \mathcal {T2}$ 第一眼四元环计数 感觉就像我改编的那道猫树分治+bitset 题，但脑抽了只往 bitset 上想了。

$\mathcal{T3}$ 只想到了二分，但不清楚怎么判定就弃掉了。这种判定方法还是要学一下。

Solution

one

原操作相当于什么？把所有的 $1$ 都看成左括号，匹配出合法的括号序列并使得所有右括号尽可能靠左。

引理：定义 $a_i$ 表示内部包含括号最大层数为 $i-1$ 的括号对数量，则对于原操作，所有的 $a_i$ 都不会改变

考虑证明引理：前一次的右括号就是后一次的左括号。也就是说这个东西向前推和向后推一步都是等价的。

也就是 $01$ 和 $10$ 是等价的。证毕。

所以最后的答案一定是一个类似后缀和的形式。

用堆每次处理最小的那个再合并（删除一段连续的 $01$，显然最小的那一段最先被删完）。

two

分治上下左右边界。

在一个矩形内，假设我们枚举上下边界，维护左右可达性，只有长的边界更新到短的。

然后维护一下是那个边界更新过来的，把左右两边方案数乘起来就好了。

three

枚举直径中心，由于中心可能在边上就在所有边上加个点（使得每条原路径现在的点数为奇数）。

假设知道直径中心，和半径 $R$，叶子个数 $x$，叶子总距离 $d$，由于我们显然要把答案加在叶子上，所以此时我们最多的添加次数为 $\lfloor \dfrac{xR-d}{2}\rfloor$，换根计算每个点的相关值。

设 $dp[i]$ 表示半径为 $i$ 时最多加多少，先计算再转移： $dp[i+2]=dp[i]+Num_of_Leaf\times2$

最后直接二分计算即可。

注意全部超出的特判，就是说你半径大于所有点还不行的情况。

左偏树学习笔记

2021-03-16
#数据结构

今天学了一下左偏树。

其实也很容易。

我们定义 $dis_i$ 。

空节点 $dis$ 值为 $0$，其他点 $dis$ 值为他右儿子的 $dis$ 值 $+1$。同时保证：$dis_{ls}\ge dis_{rs}$。

由于此性质，容易发现根节点 $dis$ 约为 $\log n$。（$n$ 为节点个数）

然后我们要合并两个点。

int Merge(int x, int y)
{
	if(!x || !y) return x | y;
	if(heap[x].val > heap[y].val) swap(x, y);
	heap[x].rs = Merge(heap[x].rs, y);
	if(heap[heap[x].rs].dis > heap[heap[x].ls].dis) swap(heap[x].ls, heap[x].rs);
	heap[x].dis = heap[heap[x].rs].dis + 1;
	return x;
}

递归合并右儿子，总复杂度 $O(\log n+\log m)$。

删除根就是合并两个儿子并清空根的信息。

删除任意节点：合并左右儿子，自底向上维护左偏性质。

全局修改什么的就是在根上打一个 $tag$，更新时向下传递即可。

好的博弈论。

由于数据提示分奇偶，所以我们根据奇偶性讨论。

容易发现奇数局面只跟奇数的奇偶性有关。

而偶数容易发现 $SG$ 函数有如下规律：

  1. $x\%(id+1)=id$，SG=2

  2. $(x\%(id+1)) \% 2=1$，SG=1

  3. $(x\%(id+1))\%2=0$，SG=0

然后两个独立游戏异或起来就好了
