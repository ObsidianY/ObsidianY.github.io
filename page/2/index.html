左偏树学习笔记

2021-03-16
#数据结构

今天学了一下左偏树。

其实也很容易。

我们定义 $dis_i$ 。

空节点 $dis$ 值为 $0$，其他点 $dis$ 值为他右儿子的 $dis$ 值 $+1$。同时保证：$dis_{ls}\ge dis_{rs}$。

由于此性质，容易发现根节点 $dis$ 约为 $\log n$。（$n$ 为节点个数）

然后我们要合并两个点。

int Merge(int x, int y)
{
	if(!x || !y) return x | y;
	if(heap[x].val > heap[y].val) swap(x, y);
	heap[x].rs = Merge(heap[x].rs, y);
	if(heap[heap[x].rs].dis > heap[heap[x].ls].dis) swap(heap[x].ls, heap[x].rs);
	heap[x].dis = heap[heap[x].rs].dis + 1;
	return x;
}

递归合并右儿子，总复杂度 $O(\log n+\log m)$。

删除根就是合并两个儿子并清空根的信息。

删除任意节点：合并左右儿子，自底向上维护左偏性质。

全局修改什么的就是在根上打一个 $tag$，更新时向下传递即可。
