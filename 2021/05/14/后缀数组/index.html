后缀数组

2021-05-14
#字符串

一直学了就忘，所以记一下。

定义

定义 $sa[i]$ 表示将所有后缀排序后第 $i$ 小的后缀的编号，$rk[i]$ 表示后缀 $i$ 的排名。

则 $sa[rk[i]]=rk[sa[i]]=k$。

后缀排序

思想

运用了倍增的方法。

首先比较第一位，相当于对每个字符排了一遍序。

接着比较下一位，这时就要用上之前已经处理过的信息。

也就是说，以第一位为第一关键字，第二位为第二关键字。

然后第一、二位为第一关键字，第三、四位为第二关键字。以此类推。

实现

这里 $s$ 表示字符串， $n$ 为字符串长度，$m$ 为字符集大小。

$x[i]$ 表示第 $i$ 个后缀当前的第一关键字，$y[i]$ 表示第二关键字排名为 $i$ 的数，第一关键字的开头的位置。

先放完整代码

void GetSA()
{
	For(i, 1, n) ++c[x[i] = s[i]];
	For(i, 2, m) c[i] += c[i - 1];
	foR(i, n, 1) sa[c[x[i]]--] = i;
	for(int k = 1; k <= n; k <<= 1)
	{
		int num = 0;
		For(i, n - k + 1, n) y[++num] = i;
		For(i, 1, n) if(sa[i] > k) y[++num] = sa[i] - k;
		For(i, 1, m) c[i] = 0;
		For(i, 1, n) ++c[x[i]];
		For(i, 2, m) c[i] += c[i - 1];
		foR(i, n, 1) sa[c[x[y[i]]]--] = y[i], y[i] = 0;
		swap(x, y);
		x[sa[1]] = 1, num = 1;
		For(i, 2, n) x[sa[i]] = (y[sa[i]] == y[sa[i - 1]] && \
                                 y[sa[i] + k] == y[sa[i - 1] + k]) ? num : ++num;
		if(num == n) return;
		m = num;
	}
    return;
}

首先对每个位置计数排序：

For(i, 1, n) ++c[x[i] = s[i]];
For(i, 2, m) c[i] += c[i - 1];
foR(i, n, 1) sa[c[x[i]]--] = i;

然后倍增：

for(int k = 1; k <= n; k <<= 1)

由于 $n-k+1 \sim n$ 这一段的第二部分为空，所以它们的关键字是最小的：

For(i, n - k + 1, n) y[++num] = i;

求出剩下的 $y$ 数组：

For(i, 1, n) if(sa[i] > k) y[++num] = sa[i] - k;

利用得到的信息排序。这里倒序枚举部分，具体来说倒序枚举第二关键字，那么在第一关键字所处的值域块中，排名也是倒序的：

For(i, 1, m) c[i] = 0;
For(i, 1, n) ++c[x[i]];
For(i, 2, m) c[i] += c[i - 1];
foR(i, n, 1) sa[c[x[y[i]]]--] = y[i], y[i] = 0;

现在我们要得出新的 $x$，所以先把现在的第一关键字用 $y$ 存着：

swap(x, y);

现在我们要利用 $sa$ 和 $y$（也就是较短的第一关键字）求出新的更长的第一关键字了。注意到 $sa$ 这里面的排序根据的长度其实已经是新的第一关键字的长度。

然后如果两个相同，就需要再次排序。

如果排出来没有重复，说明排完了。否则，字符集大小变为关键字最大值。

x[sa[1]] = 1, num = 1;
For(i, 2, n) x[sa[i]] = (y[sa[i]] == y[sa[i - 1]] && \
                         y[sa[i] + k] == y[sa[i - 1] + k]) ? num : ++num;
		if(num == n) return;
		m = num;

最长公共前缀

定义

定义 $lcp(i,j)$ 表示后缀 $sa[i]$ 和 $sa[j]$ 的最长公共前缀。

性质

1.$lcp(i,j)=lcp(j,i)$

2.$lcp(i,i)=n-sa[i]+1$

显然可知。

3.$\forall 1\le i\le j\le k\le n,lcp(i,k)=\min(lcp(i,j),lcp(j,k))$

感性证明：因为 $i\le j\le k$，所以 $sa[i]$ 与 $sa[j]$ 的前缀相似程度会更大。那么 $lcp(i,k)\le lcp(i,j),lcp(i,k)\le lcp(j,k)$

然后因为 $sa[i]$，$sa[j]$ 至少前 $\min(lcp(i,j),lcp(j,k))$ 个字符相等， $sa[j],sa[k]$ 也是。所以 $lcp(i,k)\ge \min(lcp(i,j),lcp(j,k))$

所以 $3$ 成立

4.$\forall 1\le i< j\le k\le n,lcp(i,k)=\min\{lcp(j-1,j)\}$

由 $3$，$lcp(i,k)=\min(lcp(i,i+1),lcp(i+1,k))$ 这样一直拆下去就好了。

height

定义 $height[i]=lcp(i-1,i)$。

由 $4$ 知，$lcp(i,k)=\min \{height[j],i<j\le k\}$。

设 $h[i]=height[rk[i]]$，则有 $height[i]=h[sa[i]]$。

此时有： $\texttt{h[i]}\ge \texttt{h[i-1]-1}$

code

void GetHeight()
{
    int k = 0;
    For(i, 1, n) rk[sa[i]]=i;
    For(i, 1, n)
    {
        if(rk[i] == 1) continue; 
        if(k) --k;
        int j = sa[rk[i] - 1];
        while(j + k <= n && i + k <= n && s[i + k] == s[j + k]) ++k;
        height[rk[i]] = k;
    }
    return;
}

应用

  1. 求最长公共前缀。由 $3$ 可知求两个后缀的 $lcp$ 可以转化为 RMQ 问题。

  2. 比较两子串大小。显然。

  3. 不同子串个数
